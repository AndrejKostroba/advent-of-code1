# Advent Of Code

Some python-powered solution for [Advent of Code](https://adventofcode.com/).

# Current state

* 2017 - 12/25 tasks
* 2018 - 2/25 tasks
* 2019 - 10/25 tasks
* 2020 - In progress!

# What's used

* Python 3.7
* pytest >= 5.3.1
* isort >= 4.3.21

# How to run tests

For example, you need to run tests for 2020 and day_01. Command will be:
```bash
pytest 2020/day_01
```
