# Project requirements

* Python 3.10.6 (upd. 2022-12-01)
* pytest

```bash
pip install -r requirements.txt
```

# How to create new dir for new task

```bash
./template.py
```

# How to run tests

For example, you need to run tests for 2020 and day_01. Command will be:
```bash
pytest 2021/day_01
```
